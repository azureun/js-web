## DOM 트리 - DOM 구조는 나무처럼 생겼다
> DOM 트리 웹 문서 요소 표현

    ___body__
    |        | -> (가지)
    h1       p -> (노드)
    |        |
   '제목'   '본문'

- 웹 문서의 태그는 요소(Element) 노드로 표현.
- Tag가 품고 있는 Text는 해당 요소 노드(Tag)의 자식 노드인 Text 노드로 표현.
- Tag의 속성은 모두 해당 요소 노드(Tag)의 자식 노드인 속성(Attribute) 노드로 표현
- 주석은 주석(Comment) 노드로 표현.
